# Site de Portfólio.

Bem-vindo ao repositório do meu site de portfólio. Este projeto tem como objetivo apresentar meus projetos, habilidades e informações de contato de forma profissional e visualmente atraente.

## Estrutura do Projeto

- **index.html**: Página inicial do site contendo uma breve introdução e navegação para outras seções.
- **assets/**: Diretório contendo todos os arquivos de mídia, como imagens e PDFs.
  - **images/**: Imagens utilizadas no site.
  - **js/**: Arquivos JavaScript para funcionalidades adicionais.
- **algoritimos/**: Scripts e arquivos relacionados a projetos específicos que demonstrem habilidades em algoritmos.
- **node_modules/**: Módulos Node.js usados no desenvolvimento do site.
- **server.js**: Arquivo de configuração para rodar o site em um servidor local.
- **README.md**: Este arquivo com informações sobre o projeto.

## Funcionalidades

- **Seção Sobre Mim**: Informações pessoais e profissionais.
- **Projetos**: Uma galeria dos projetos desenvolvidos, com descrições e links para os repositórios.
- **Currículo**: Um link para download do meu currículo em PDF.
- **Contato**: Formulário de contato para visitantes entrarem em contato diretamente comigo.

## Tecnologias Utilizadas

- **HTML5**: Estruturação do conteúdo do site.
- **CSS3**: Estilização e layout do site.
- **JavaScript**: Funcionalidades dinâmicas e interatividade.
- **Node.js**: Execução do site em um servidor local.
- **Express**: Framework para simplificar a configuração do servidor.
